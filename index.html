HID Project Sheet Clipper — Privacy Policy

Last updated: [DATE]

HID Project Sheet Clipper is a browser extension that lets a user capture product details, screenshots, and links from websites they visit and save them as items in their organization's HID Projects Portal.

What the extension accesses

The extension only accesses information when the user actively uses it. It does not run in the background or monitor browsing.

  • Page content the user selects: the specific text, images, screenshots, and the page URL and title the user chooses while using the capture tools.
  • Authentication: the extension signs the user in through Microsoft Entra ID and receives a short-lived access token to authorize calls to the user's portal.

How information is used

The captured product details, screenshots, and link are sent only to the user's own HID Projects Portal (the address the user configures in the extension's options), solely to create the project item the user requested. The access token is used only to authorize those calls.

What the extension does not do

  • It does not sell, rent, or share data with any third party.
  • It does not use data for advertising, analytics, or profiling.
  • It does not transmit data to any server other than the user's configured HID Projects Portal and Microsoft's sign-in service.
  • It does not collect browsing history or read pages the user has not acted on.

Storage and retention

The extension stores its configuration (the selected environment and portal URL) locally in the browser. The access token is held only in temporary in-memory session storage and is cleared when the browser closes; no long-lived refresh token or password is stored. Submitted product data resides in the HID Projects Portal, governed by the organization's own data policies.

Authentication

Sign-in is handled entirely by Microsoft Entra ID. The extension never sees or stores the user's Microsoft credentials.
